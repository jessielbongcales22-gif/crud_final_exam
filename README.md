# 🎓 Student Information Management System

A simple CRUD (Create, Read, Update, Delete) web application built using **Node.js, Express, MySQL, HTML, CSS, and JavaScript**.

This system allows users to manage student records efficiently through a web interface.

---

## 🚀 Features

- ➕ Add new student records
- 📋 View all students in a table
- ✏️ Update student information
- ❌ Delete student records
- 🎨 Clean and responsive UI design
- 🔗 REST API integration

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MySQL
- HTML5
- CSS3
- JavaScript (Vanilla)
- dotenv
